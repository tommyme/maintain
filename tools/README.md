# tools 编程规范

这个文件夹是加入全局变量里面的，随着工具种类的增加，我们越来越不容易记住我们创建过的工具，所以我们要有一个规范来更高效的使用自己创建的工具

## api-key类型的工具

这类工具一定不要把key上传到github上面

## 工具编程规范
开发的时候可以写单独的脚本

最后的时候还是汇聚在一起比较好

依赖就是 click和rich，然后使用脚本来安装依赖
